# Book-catalog
Графическое приложение для ведения личного каталога книг. Позволяет: - Добавлять книги (название, автор, жанр, оценка 1-5, статус чтения) - Редактировать выбранную книгу - Удалять книги - Фильтровать книги по статусу (Все / Прочитана / Читаю / Брошена / В планах) - Данные сохраняются в файл books.json
