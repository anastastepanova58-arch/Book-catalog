---

## Описание программы

*Книжный каталог — это графическое приложение для ведения личной библиотеки. Позволяет добавлять книги (название, автор, жанр, оценка от 1 до 5, статус чтения), редактировать и удалять записи, фильтровать книги по статусу (Прочитана, Читаю, Брошена, В планах). Все данные автоматически сохраняются в JSON-файл.*

---

## Требования для запуска

Что нужно установить на компьютер:

- Python 3.10 или выше
- Дополнительные библиотеки не требуются (tkinter входит в стандартную поставку Python)
